<h1>Веб-приложение React.js. TODO-List.</h1>

## Использованные технологии

- [React.js](https://reactjs.org)

## Функционал

- Просмотр списка задач
- Добавление задачи (название, выполнено/не выполнено, дата (опционально))
- Изменение задачи
- Удаление задачи
